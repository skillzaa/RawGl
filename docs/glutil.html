GlUtil Object

The GlUtil is a low level object that bundles common functions to abstract out WebGl boiler code.

The user is not required to use this directly but we need this object to atleast obtain the WebGl drawing context and clear canvas.

This object just export static functions and should not be instantiated.

The main two functions that we will use are:

  1. static getGl(canvasId :string ="bilza"):WebGL2RenderingContext
  2. static clear(gl :WebGL2RenderingContext,r:number=0,g:number=0,b:number=0,a:number=1
