###### started at version 0.0.1 ## Core Components RawGL provides you with 3 core components which shall do all the drawing. These components are: - BaseComp (Base Component). - TriComp (Tri Component) - TriContComp (Tri Container Component)

All the above three components has just one purpose and that is to draw Triangles on the canvas (which is the main objective of webgl).

All of the above Components (component is called comp in these documents) implements IGPUComp interface. It is called GPU comp since (in WebGl) the code runs in GPU and not on CPU.

Common features of All 3 Comps

All the components at the time of creation does not hold any data to draw.
