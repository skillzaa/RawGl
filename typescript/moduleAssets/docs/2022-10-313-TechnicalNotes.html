Asset Builder Version Zero

The Concept of Assets Module:

  1. We have simple shapes (called primtives) for example rectangle, triangle etc. We can compose larger images and assets from these smaller shapes. The AssetBuilder object help us in creating assets.
  2. The second purpose of assets module is to provide ready to consume assets.
  3. Every asset has x , y , width and height.
  4. Other than this it has a pallete. The user can use this pallate.

The AssetCollection class has static functions which will return an asset builder object. We can change its properties and then extract an asset out of it.
